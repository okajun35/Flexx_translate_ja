PyScript quick user guide
=========================

This guide demonstrates the features that PyScript supports and how it
maps to JavaScript. In most cases you can just read what is
:ref:`not supported <pyscript-support>` and the
:ref:`caveats <pyscript-caveats>` and start coding.
