Welcome to Flexx’s documentation!¶

Flexxは、グラフィカルユーザーインターフェイス（GUI）を作成するための純粋なPythonツールキットで、レンダリングにWebテクノロジを使用します。アプリは純粋にPythonで書かれています。 Flexxのtranspilerは必要なJavaScriptをすぐに生成します”

You can use Flexx to create (cross platform) desktop applications, web applications, and (if designed well) export an app to a standalone HTML document. It also works in the Jupyter notebook.

ドキュメントは Readthedocs に、コードは Github にあり、デモサーバは AWS と Raspberry pi にあります。 Flexxは現在アルファステータスです。APIの一部は予告なしに変更されることがあります。

Flexxにはモジュール式の設計でいくつかのサブパッケージで構成されており、単体でも使用できます。以下の目次を参照してください。アプリケーションの構築にはこれら3つのモジュールが最も関連性があります：

  • flexx.ui - widget (読む intro, api, 例)

  • flexx.app - イベントループとサーバー (読む intro, api, 例)

  • flexx.event - プロパティとイベント (読む intro, api, 例)
